# Task Manager System

## 🚀 Features

* Add, update and delete tasks
* Simple user-friendly interface
* Task tracking system

## 🛠 Tech Stack

* Python (Flask)
* HTML
* CSS

## ▶️ How to Run

1. Install dependencies
   pip install flask

2. Run the app
   python app.py

3. Open browser
   http://127.0.0.1:5000/

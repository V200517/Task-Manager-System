# 📝 Task Manager System

A simple web-based Task Manager built using **Python (Flask)** with **HTML & CSS** for frontend.

---

## 🚀 Features

* Add new tasks
* Display task list
* Simple and clean UI
* Lightweight and easy to use

---

## 🛠 Tech Stack

* Python (Flask)
* HTML
* CSS

---

## ▶️ How to Run

1. Install Flask
   pip install flask

2. Run the application
   python app.py

3. Open in browser
   http://127.0.0.1:5000/
